Basic Operations

                    
1. Drag blocks from the left toolbox to the middle workspace

                    
2. Connect blocks to build a machine learning workflow

                    
3. Click "Run Model" to execute training

                    
4. View output results and visualizations on the right

                    
                    
Block Descriptions

                    
Data Processing: Load datasets, split data, normalize, etc.

                    
Model Building: Create linear models, neural networks, etc.

                    
Training & Evaluation: Train models, evaluate performance, predict

                    
Visualization: Plot data, training history, confusion matrix, etc.
